# Study Planner Agent

An AI-powered study planner built using the Google Agent Development Kit (ADK) to help students generate personalized study schedules.

## Features

- Personalized study schedule generation
- Planner Agent for creating study plans
- Modular architecture for future multi-agent expansion
- Simple web interface for interacting with the planner

## Tech Stack

- Python
- Google Agent Development Kit (ADK)
- Gemini API
- HTML

## Project Structure

```
agent.py
minimal_ui.html
requirements.txt
test_schemas.py
```

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/debojyotiroy062007-ctrl/Study-Planner-Agent.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Configure your Gemini API key in a `.env` file

```
GEMINI_API_KEY=YOUR_API_KEY
```

4. Run the project using Google ADK.
